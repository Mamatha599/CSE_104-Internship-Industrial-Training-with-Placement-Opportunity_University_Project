# AI-Driven Internship & Placement Management System

A comprehensive university capstone project for managing internships and placements using AI.

## Project Structure

- `backend/`: Node.js + Express API
- `frontend/`: React + Vite Frontend
- `docs/`: Project documentation

## Getting Started

1. Clone the repository.
2. Follow the README in the `backend/` and `frontend/` directories for setup.
